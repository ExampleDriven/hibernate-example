hibernate-example
=================

This is a JPA 2 Example application, using Hibernate as the persistence provider. It demonstrates the use of the following technologies : 

- HSQL in memory DB
- Hibernate 4.0
	- Database generation
	- @Formula
	- @Cacheable
	- @Cache
- JPA 2
	- @JoinColumn
	- @Id
	- @Column
	- @GeneratedValue
	- @Entity
	- @NamedQuery
- Spring
	- @Transactional
- EHCache


To start the project type mvn install
